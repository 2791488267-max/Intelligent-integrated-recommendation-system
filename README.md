## 📖 项目简介
当前非遗文化（如南华县彝绣）产品面临“酒香也怕巷子深”的困境。传统电商平台依赖基于历史销量的“协同过滤”算法，导致小众非遗产品遭遇严重的“头部品牌偏见”。同时，传统平台依赖关键词硬匹配，无法理解消费者（如“想送长辈一件寓意健康长寿的礼物”）深层的自然语言文化诉求。

本项目旨在开发一款具备自然语言深度理解与深层语义交互能力的文化电商 Web 平台——**南华彝绣数字赋能平台**。我们致力于成为“懂彝绣的文化科技伙伴”，为您提供“可解释的智能推荐”，让每一次购买都成为一次深度的非遗文化体验。

## 🚀 核心技术创新：轻量化 RAG 架构降维打击
本项目在核心技术架构上进行了极具工程思维的敏捷重构：
1. **摒弃算力堆砌**：彻底放弃了极度消耗内存的图数据库（Neo4j）与庞大的本地向量模型（Sentence-BERT），解决了在入门级云服务器上极易发生内存溢出（OOM）的痛点。
2. **自研轻量级检索算法**：采用 Pandas 构建本地“微型结构化知识库”，结合自研的**多维字符串滑动窗口特征匹配算法（Bi-gram 加权思想）**，实现了毫秒级的高置信度实体召回。
3. **国产大模型深度赋能**：将召回的绝对准确的领域知识（Ground Truth）作为上下文注入**国产智谱 GLM 大语言模型**，彻底规避了通用大模型的“知识幻觉”，生成兼具文化底蕴与情感温度的推荐语。

## 🛠️ 技术栈 (Tech Stack)
* **前端表现层**：HTML5 / CSS3 / Vanilla JS
  * *UI 特性*：现代 CSS 变量、Flexbox 响应式瀑布流布局、毛玻璃 (Glassmorphism) 高级视觉特效、无刷新状态管理（购物车与个人中心面板）。
* **后端服务层**：Python + Flask (提供 RESTful API 接口，跨域资源配置)
* **数据检索层**：Pandas 内存级数据分析引擎
* **AI 智能引擎**：智谱 AI 商业级大语言模型 API (GLM-4)

## 💻 本地开发与二次部署指南
如果您希望在本地开发机上运行本系统源码，请遵循以下极简步骤：

### 1. 环境准备
确保您的计算机已安装 Python 3.8 或更高版本。
克隆或下载本仓库源码后，在项目根目录打开终端，执行以下命令安装依赖：
```bash
pip install Flask pandas requests openpyxl
```

### 2. 配置大模型 API 密钥
1. 访问 [智谱 AI 开放平台](https://open.bigmodel.cn/) 注册并获取您专属的 API Key。
2. 使用编辑器打开本项目后端的 `rag_with_qwen.py` 文件。
3. 找到代码中的大模型请求配置部分，将您的 API Key 填入指定位置。
   *(注意：请妥善保管您的密钥，切勿将真实明文密钥提交到公开的 GitHub 仓库中！)*

### 3. 启动服务
在终端中执行以下命令启动 Flask 开发服务器：
```bash
python app.py
```
启动成功后，在浏览器中访问 `http://127.0.0.1:5000` 即可在本地体验智慧融合推荐系统。

*(注：如果在 Linux 云服务器环境部署，建议使用 `nohup python3 app.py > output.log 2>&1 &` 指令以守护进程模式后台运行。)*
